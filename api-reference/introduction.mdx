## Base URL

```
https://api.solmarket.app/api/v1
```

<Note>
  The SolMarket API is used by the frontend to fetch market data, user positions, and submit metadata. **All financial transactions happen directly on Solana** — the API never handles funds.
</Note>

## Endpoints

### Markets

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/markets` | List all markets (active, resolved, refunded) |
| `GET` | `/markets/:id` | Get a single market with pool data and activity |
| `POST` | `/markets` | Create a new market (admin only) |

### Users

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/users/:wallet/bets` | Get all bets for a wallet address |
| `GET` | `/users/:wallet/claims` | Get all claims for a wallet address |

### Admin

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/admin/resolve` | Trigger market resolution (oracle) |
| `POST` | `/admin/create-market` | Create a new prediction market |

## Authentication

Admin endpoints require a Bearer token. Public endpoints (markets, user data) are read-only and unauthenticated.

```bash
# Example: Fetch all active markets
curl https://api.solmarket.app/api/v1/markets

# Example: Fetch user bets
curl https://api.solmarket.app/api/v1/users/YOUR_WALLET/bets
```

## Response format

All responses return JSON:

```json
{
  "data": [...],
  "error": null
}
```

Error responses:

```json
{
  "error": "Market not found"
}
```
